# Running Instructions
You do not need to perform these steps as both frontend and backend are already deployed at the following locations
1. Frontend - https://kollabus-technosharks.netlify.app/
2. Backend - https://collabus-backend.onrender.com/

#### Steps to run project locally
1. Clone this repository on your local system.
2. Open the client folder in terminal and run <code>npm install</code>
3. Start the client (frontend) using <code>npm start</code>
4. Open the server folder in terminal and run <code>npm install</code>
5. Start the server (backend) using <code>npm start</code>
